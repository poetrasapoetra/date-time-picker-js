DateTimePickerOption

Type:
  • object
Properties:
Name Type Attributes Description
start_date Date <optional>
 Start of picker date available. Default `new Date('0001-01-01')`
end_date Date <optional>
 End of picker date available. Default Infinity
selected_date Date <optional>
 Selected date
theme_class string <optional>
 Class of picker theme. Default "" light theme, date-time-picker also has dark theme that can be activate by set this to "dark"
locale DateTimePickerLocale <optional>
 Your custom locale
show boolean <optional>
 If true then picker will showed after initialization
show_today_button boolean <optional>
 Should today button showed?. This button will select today. Default false
close_on_select boolean <optional>
 Picker will close after value selected, otherwise select and cancel button will show. Default true
type "date" | "month" | "year" | "datetime-local" <optional>
 Picker type, default "date"
first_day number <optional>
 Day name to show in first column. Only available if type is "date" or "datetime-local". 0: Subday, 1: Monday, 6: Saturday
theme DateTimeCustomTheme <optional>
 Your custom theme, set inline style means all style in theme_class thet overlap to this value will not be respected
